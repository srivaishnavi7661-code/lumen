## üõ†Ô∏è Installation and Setup

Follow these steps to get a local copy of Lumen up and running on your machine.

### Prerequisites

* **Node.js** (version 16 or higher recommended)
* A **Gemini API Key** (available from [Google AI Studio](https://ai.google.dev/))

### Steps to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/srivaishnavi7661-code/lumen.git](https://github.com/srivaishnavi7661-code/lumen.git)
    cd lumen
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Set up the API Key:**
    Create a file named `.env.local` in the root directory of the project and add your Gemini API key:
    ```
    GEMINI_API_KEY="YOUR_API_KEY_HERE"
    ```

4.  **Run the application:**
    ```bash
    npm run dev
    ```

The application will now be running, typically at `http://localhost:5173`.

## üíª Usage

Once the application is running, navigate to the local host URL in your browser:

1.  **Input your details:** The interface will prompt you to enter information about your current skills, interests, education, or desired career direction.
2.  **Generate a path:** Click the "Find Career Path" or similar button.
3.  **Review the results:** The AI (powered by Gemini) will process your input and present a structured output detailing potential career roles, required steps, and suggested learning materials.
